# Fonts

This folder contains custom fonts for the Kindling app.

## Required Fonts

### Montserrat-SemiBold.ttf

Used for the splash screen tagline "Protect today. Build tomorrow."

**Download from Google Fonts:**
1. Go to https://fonts.google.com/specimen/Montserrat
2. Download the font family
3. Extract and copy `Montserrat-SemiBold.ttf` to this folder

**Or use this direct link:**
https://fonts.google.com/download?family=Montserrat

After downloading, ensure the file is named exactly: `Montserrat-SemiBold.ttf`
